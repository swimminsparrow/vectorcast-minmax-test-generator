5. Verificare Risultati di esecuzione

Selezionare dal Menu “Run” → “Verify Execution Results”.

Se è stato selezionato un Environment corretto al passo 2. La vista sarà popolata con tutte le informazioni necessarie relative ai test presenti nell’Environment.

Alcune informazioni mostrate relative al TestCase:

  • Environment

  • UUT

  • SubProgram

  • TestCase Name

  • Numero Parametri Input

  • Execution Result

  • Execution Date

  • Elenco Input per ogni singolo TestCase



La tabella mostra le seguenti informazioni:

  • TestCase: nome TestCase alberato secondo l’ordine UUT, SubProgram, TestCase.

  • Expected Result: risultato atteso in base al nome del TestCase.

  • Result: risultato esecuzione del TestCase in VectorCast.

  • Matching: POSITIVO se Expected Result = Result, NEGATIVO altrimenti.
