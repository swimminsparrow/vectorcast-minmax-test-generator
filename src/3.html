3. Configurazione Parametri

Al termine della parserizzazione dei testcase, l’interfaccia principale mostra un elenco dei parametri già presenti nel db configurati in precedenza. Selezionando un parametro saranno mostrate ulteriori informazioni nel pannello “Param Details”.

Per aggiungere un parametro alla lista cliccare su “Edit/Add”. Esistono due possibilità.

Default Global Values

Consente di specificare un set di valori globali per il singolo parametro:

  • Locked Value: un valore immutabile per i test Min e Max.

  • [MIN, MAX]: un messaggio di errore comparirà se sarà rilevato un valore di Min >= Max. Inoltre, in caso di errore di digitazione (es. valore non inserito) il valore sarà automaticamente inizializzato a 0.

Override Values for a specified Function

Se desideri effettuare un override di un set di valori di default per una specifica funzione puoi selezionarne una dal menu a tendina.

Function Values

Riporta la lista delle funzioni che che dichiarano nell’header il parametro selezionato.

Custom Values per Function

Mostra l’elenco dei set valori specificati per le singole funzioni.
