2. Importare parametri input delle funzioni

Dal Menu “Run” selezionare “Run Parser”.

In via facoltativa è possibile scegliere per quali file “.c” generare i test utilizzando il campo Select component source file path ed eliminare le funzioni void func(void) da questa fase.

È possibile importare i parametri di input delle funzioni nel tool in 2 modi:

  1. (Automatic) Environment Path: selezionare path dell'environment VectorCast da cui si intende importare automaticamente i parametri. Verificare che il path contenga il file NomeEnvironment.vce

  2. (Manual) TST: selezionare il TST relativo all’Environment VectorCast esportato manualmente.
