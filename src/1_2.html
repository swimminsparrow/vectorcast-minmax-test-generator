1.2 - Impostazioni Tool

Verifica i parametri di configurazione in "Edit” → “Settings".

VectorCast Installation Folder:

Path cartella di installazione di VectorCast. Il path deve corrispondere con la posizione dell’eseguibile clicast.exe


VectorCast License Server Start Script:

Inserire lo script per avviare il server della licenza di VectorCast utilizzato sul proprio ambiente. Di solito dovrebbe essere necessario semplicemente verificare i path specificati nello script. In caso contrario modificare lo script con estrema cautela. Le funzionalità che prevedono l’interfacciamento diretto con l’ambiente VectorCast potrebbero non funzionare correttamente in caso di problemi di esecuzione del server di licenza.
