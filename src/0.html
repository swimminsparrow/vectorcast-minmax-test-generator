Informazioni Generali:

Il Tool consente di eseguire le seguenti operazioni:

  • Gestire in automatico tutti i parametri numerici di input alle funzioni di un Environment VectorCast specificato dall’utente.

  • Generare Test di Boundary e importarli nell’Environment VectorCast Specificato.

  • Verificare i risultati di esecuzione dei TestCases.



L’icona del tool indica le funzionalità che prevedono l'interfacciamento con VectorCast.
